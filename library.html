Przydatne linki i inne dodatkowe materiały

Linki do interesujących materiałów:

HTML Validator

Strona służąca do sprawdzenia czy kod html jest poprawy semantycznie (Sprawdza m.in czy w sekcjach/artykułach nie brakuje nagłówków oraz czy nie zapomnieliśmy domknąć jakiegoś tagu)

CSS Diner - flukeout

Gra która służy do ćwiczenia różnych własności CSSowych. Dobre narzędzie do poćwiczenia celem utrwalenia wiedzy.

htmlpreview github

Strona na której możemy wrzucić link do pliku .html z repozytorium githubowego żeby wyrenderować podgląd załączonego pliku w przeglądarce

Wyjaśnienie baseline alignment

Artykuł zawierający przykłady działania własności baseline. Duże ułatwienie w zrozumieniu czego oczekiwaćo od tego ustawienia.

Flexbox froggy

Gra która służy do ćwiczenia różnych własności flexa. Jej celem jest ustawianie własności planszy/żaby w taki sposób, żeby żaby znajdowały się na liściach odpowiednych kolorów. Dobre narzędzie do poćwiczenia celem utrwalenia wiedzy.

Codepen

Strona pozwalająca na szybko zamieścić kod html/css/js w celu przetestowania czegoś lub szybkiego zasymulowania/udostępnienia jakiegoś przykładu.
